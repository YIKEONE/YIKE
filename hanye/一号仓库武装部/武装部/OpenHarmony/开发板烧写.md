烧机方法：

下载rk3568_loader_for_linux.tar.gz文件 [rk3568_loader_for_linux.tar.gz](https://neusoft.feishu.cn/file/CAaMbHPJ7osQjDxovgRc11AMnhd)

在代码根目录解压，解压后生成“linux”目录

执行 sudo cp linux/etc/udev/rules.d/85-rk3568.rules /etc/udev/rules.d/

重启udev服务，执行 sudo service udev restart

DAYU200开发板介绍：[](https://security.feishu.cn/link/safety?target=https%3A%2F%2Fgitee.com%2Fopenharmony%2Fdevice_board_hihope%2Ftree%2Fmaster%2Frk3568&scene=ccm&logParams=%7B%22location%22%3A%22ccm_mindnote%22%7D&lang=zh-CN)[https://gitee.com/openharmony/device_board_hihope/tree/master/rk3568](https://gitee.com/openharmony/device_board_hihope/tree/master/rk3568)

将开发板USB线连接PC，按住recovery按键，然后开发板上电，然后松开recovery按键，执行sudo lsusb命令，查看开发板是否进入烧写模式，如下图：

![Untitled](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/64f9fdac-a2e2-4213-8f1e-f14bae24b8f5/Untitled.png)

确认进入烧写模式后，在代码根目录执行命令：python linux/flash.py -a 将执行开发板烧写过程

rk3568板子烧写成功之后，连接WiFi网络

设置HiStreamer，hdc shell param set debug.media_service.histreamer 1

设置hilog等级，hdc shell hilog -b D

设置屏幕延时灭屏，hdc shell power-shell timeout -o 40000000

安装测试hap包，hdc install ActsHLSPlayerRustJsTest.hap

截取log，hdc shell hilog > {log路径}/hilog.log

点击安装成功的hap包
